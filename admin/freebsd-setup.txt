# Лог установки и настройки FreeBSD 9.1 на Toshiba Portege с core i7

Установка по хендбуку FreeBSD серии 9.х с образа FreeBSD-9.1-RELEASE-amd64-memstick.img

Сначала - как загрузиться с флешки вообще!
Зайти в БИОС - по F2
В биосе по умолчанию стоят богомерзкие настройки: secure boot + UEFI mode, из-за чего можно загрузиться только с того, что подписано ключом, и только с UEFI-совместимого
Выключаем нафиг

Разметка диска в GPT, дальше стандартный layout для фряхи из мана
Беспроводную карточку не нашёл при настройке сети
